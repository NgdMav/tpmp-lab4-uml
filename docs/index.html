О проекте

Проект «Автомастерские» представляет собой консольное приложение на языке C для учета работы сети автомастерских.

Основные возможности:

  • Учет мастерских (адрес, телефон, специализация по маркам)
  • Учет мастеров и их специализации
  • Регистрация автомобилей и их владельцев
  • Оформление заказов на ремонт
  • Формирование отчетов по мастерским и мастерам
  • Расчет выручки и аналитика
