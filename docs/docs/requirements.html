Текстовые сценарии

Сценарий UC-08: Оформление ремонта

Актёры: Мастер (основной), Клиент (второстепенный)

Предусловия: Мастер аутентифицирован, клиент обратился в мастерскую

Основной поток:

  1. Мастер выбирает пункт меню "Оформить ремонт"
  2. Система запрашивает госномер автомобиля
  3. Мастер вводит госномер
  4. Система проверяет наличие автомобиля в БД
  5. Если автомобиль найден, система отображает информацию об авто
  6. Если автомобиль не найден, система предлагает ввести данные нового авто
  7. Мастер вводит данные нового автомобиля
  8. Система сохраняет данные автомобиля
  9. Система отображает список доступных видов работ
  10. Мастер выбирает виды работ
  11. Система рассчитывает предварительную стоимость
  12. Мастер подтверждает создание заказа-наряда
  13. Система сохраняет заказ-наряд и присваивает ему номер

Постусловия: Создан новый заказ-наряд со статусом "В работе"

Сценарий UC-03: Отчет по мастерской за период

Актёр: Заведующий

Основной поток:

  1. Заведующий выбирает пункт меню "Отчет по мастерской"
  2. Система запрашивает номер мастерской
  3. Заведующий вводит номер мастерской
  4. Система проверяет существование мастерской
  5. Система запрашивает начальную и конечную даты
  6. Заведующий вводит даты
  7. Система выполняет запрос к БД
  8. Система отображает отчет с группировкой по мастерам
