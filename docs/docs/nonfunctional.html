Общие функциональные требования

Работа с датами

  • Даты хранятся в формате ISO (YYYY-MM-DD)
  • Поддерживаются форматы ввода: ДД.ММ.ГГГГ и ГГГГ-ММ-ДД

Валидация данных

  • Госномер проверяется по регулярному выражению
  • Стоимость ремонта должна быть положительным числом
  • Дата завершения не может быть раньше даты начала
