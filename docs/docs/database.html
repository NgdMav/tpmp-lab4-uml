Схема базы данных

ER-диаграмма

Описание таблиц

Таблица workshops (Мастерские)

ПолеТипОграниченияОписание
idINTEGERPRIMARY KEY AUTOINCREMENTНомер мастерской
addressTEXTNOT NULLАдрес
phoneTEXTКонтактный телефон
car_brandsTEXTПеречень марок

Таблица mechanics (Мастера)

ПолеТипОграниченияОписание
idINTEGERPRIMARY KEY AUTOINCREMENTТабельный номер
nameTEXTNOT NULLФИО
specializationTEXTСпециализация
workshop_idINTEGERFOREIGN KEY REFERENCES workshops(id)ID мастерской

Таблица cars (Автомобили)

ПолеТипОграниченияОписание
license_plateTEXTPRIMARY KEYГосномер
brandTEXTNOT NULLМарка
modelTEXTМодель
yearINTEGERГод выпуска
owner_nameTEXTNOT NULLФИО владельца
passport_numberTEXTUNIQUEНомер техпаспорта
owner_addressTEXTАдрес владельца

Таблица repair_types (Виды работ)

ПолеТипОграниченияОписание
idINTEGERPRIMARY KEY AUTOINCREMENTID вида работ
nameTEXTNOT NULLНазвание
categoryTEXTКатегория
base_priceREALБазовая стоимость

Таблица repairs (Ремонты)

ПолеТипОграниченияОписание
idINTEGERPRIMARY KEY AUTOINCREMENTНомер заказа-наряда
workshop_idINTEGERFOREIGN KEYID мастерской
mechanic_idINTEGERFOREIGN KEYID мастера
car_licenseTEXTFOREIGN KEYГосномер авто
repair_type_idINTEGERFOREIGN KEYID вида работ
start_dateTEXTNOT NULLДата поступления
end_dateTEXTДата завершения
costREALNOT NULLСтоимость
statusTEXTDEFAULT 'in_progress'Статус

Связи между таблицами

  • workshops (1) — (M) mechanics: Одна мастерская → много мастеров
  • workshops (1) — (M) repairs: Одна мастерская → много ремонтов
  • mechanics (1) — (M) repairs: Один мастер → много ремонтов
  • cars (1) — (M) repairs: Один автомобиль → много ремонтов
  • repair_types (1) — (M) repairs: Один вид работ → много ремонтов
